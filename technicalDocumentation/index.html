Strings

Besides numbers, Python can also manipulate strings, which can be expressed in several ways. They can be enclosed in single quotes ('...') or double quotes ("...") with the same result 2. \ can be used to escape quotes:

>>> 'spam eggs' # single quotes
'spam eggs'
>>> 'doesn\'t' # use \' to escape the single quote...
"doesn't"
>>> "doesn't" # ...or use double quotes instead
"doesn't"
>>> '"Yes," they said.'
'"Yes," they said.'
>>> "\"Yes,\" they said."
'"Yes," they said.'
>>> '"Isn\'t," they said.'
'"Isn\'t," they said.

reading files

python allows use to read files on your system very cool right

writing files

python allows us to write to a file

  • if the file does not exist python automatically creates a new file
  • if the file exists it is overwritten.
>>>myfile =open("food.txt","w") # does not need to exist but if it does overwritten
>>>myfile.write("meat") # writes content to a file
>>>myfile.close()
